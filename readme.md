Tugas2 API adalah RESTful API sederhana yang dibuat menggunakan Express.js dan MongoDB Atlas untuk mengelola data tugas.
API ini mendukung operasi CRUD (Create, Read, Update, Delete).

Project ini dibuat sebagai latihan implementasi backend menggunakan:

Node.js

Express.js

MongoDB Atlas

Mongoose

Postman untuk testing API

Fitur
✅ Create data tugas

✅ Read all data

✅ Read data by ID

✅ Update data

✅ Delete data

Tech Stack yang digunakan
Node.js

Express.js

MongoDB Atlas

Mongoose

Postman
